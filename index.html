১. প্লেইনটেক্সট, এনকোডিং, এনক্রিপশন ও হ্যাশিং


message = "hello world"  # প্লেইনটেক্সট

import base64
encoded = base64.b64encode(message.encode())
print(encoded)  # b'aGVsbG8gd29ybGQ=' (এনকোডেড)

from cryptography.fernet import Fernet
key = Fernet.generate_key()
cipher = Fernet(key)
encrypted = cipher.encrypt(message.encode())
print(encrypted)  # এনক্রিপ্টেড

decrypted = cipher.decrypt(encrypted).decode()
print(decrypted)  # ডিক্রিপ্টেড

import hashlib
hashed = hashlib.sha256(message.encode()).hexdigest()
print(hashed)  # হ্যাশ

লাইন-বাই-লাইন বিশ্লেষণ:

message = "hello world"
আসল বার্তা, যেটা পাঠাতে চাই (Plaintext)। খাঁচা পাখির মত যা যেমন আছে তেমনই।

message.encode()
স্ট্রিং কে বাইটসে রূপান্তর। বাংলা শব্দকে ইংরেজিতে ট্রান্সলিটারেট করা।

base64.b64encode(...)
ডাটাকে সাজিয়ে পাঠযোগ্য রূপে রূপান্তর। জামা বদলানো, গোপন কিছু না।

Fernet.generate_key()
এনক্রিপশনের জন্য গোপন চাবি তৈরি। গোপন বাক্সের চাবি।

cipher.encrypt(...)
বার্তাকে লুকিয়ে ফেলা। বাক্সে বার্তা ঢোকানো, চাবি ছাড়া কেউ খুলতে পারবে না।

cipher.decrypt(...)
বার্তা আবার পড়ার উপযোগী করা। চাবি দিয়ে বাক্স খোলা।

hashlib.sha256(...).hexdigest()
বার্তার একমুখী ফিঙ্গারপ্রিন্ট তৈরি, যা থেকে আসল বার্তা পাওয়া যায় না। আঙুলের ছাপ, যা দিয়ে আঙুল পাওয়া যায় না।

সংক্ষেপে সম্পর্কিত টেবিল:

বিষয় কাজ সম্পর্ক উদাহরণ
প্লেইনটেক্সট আসল বার্তা অন্যান্য সবকিছুর ইনপুট খাঁচা পাখি
এনকোডিং সাজানো, পাঠযোগ্য রূপ নিরাপত্তা দেয় না জামা বদলানো
এনক্রিপশন গোপন রাখা, কী দিয়ে খুলতে হয় নিরাপদ, কী ছাড়া খুলবে না গোপন বাক্সের চাবি
হ্যাশিং একমুখী ফিঙ্গারপ্রিন্ট ডিক্রিপ্ট হয় না আঙুলের ছাপ

২. পাসওয়ার্ড ও পাসওয়ার্ড পলিসি

পাসওয়ার্ডের উদাহরণ: dadada, 123456, cute@123

সহজ পাসওয়ার্ড দ্রুত ক্র্যাক হয়, শক্তিশালী পাসওয়ার্ড মনে রাখা কঠিন।

  • লম্বা ও জটিল পাসওয়ার্ড ব্যবহার করুন।
  • বড়/ছোট হাতের অক্ষর, সংখ্যা ও স্পেশাল ক্যারেক্টার মিশ্রিত করুন।
  • নিয়মিত পাসওয়ার্ড পরিবর্তন ও মাল্টি-ফ্যাক্টর অথেনটিকেশন ব্যবহার করুন।

@app.after_request
